# homebrew-treelines

Homebrew tap for installing `treelines`.

`treelines` is a local code-intelligence CLI for Go, Python, and Rust.

## Install

```bash
brew tap rpcarvs/treelines
brew install treelines
```

## Upgrade

```bash
brew update
brew upgrade treelines
```
